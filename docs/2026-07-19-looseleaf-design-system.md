# ルーズリーフ・デザインシステム方針

作成: 2026-07-19
状態: 主方向として採用。v80でスキン機能を停止し、v89で設定から日本語・英語を切り替え可能にした。

## 採用判断

ルーティンノートを「ノート風に装飾する」のではなく、情報構造そのものをルーズリーフとして扱う。

- 技: 一枚ずつ蓄える紙片
- ルーティン: 技の紙片を順番に束ねたもの
- 音源: 紙片を置く時間の罫線
- 練習記録・分析: 試したあとに残る書き込み跡
- 並び替え: 紙片を持ち上げて綴じ直す操作

ホームは束の表紙として扱い、機能構成は変えない。

1. アプリ名、使い方、設定
2. ルーティン練習をする
3. 前回のルーティン
4. 最下部の技ライブラリ、音源ライブラリ

## 視覚トークン

- 紙: 温かい生成り。純白にせず、日常的な道具の温度を残す
- インク: 青みのある墨色。高級文具ではなく実用品の精度を出す
- 罫線: 情報を増やさない低コントラストのガイド
- オレンジ: 練習開始と身体的な目印
- 青灰タブ: 技・並び替え
- 錆色タブ: 音源
- 影: ボタンが押せること、紙片が持ち上がることを示す短い物理フィードバック

## 全画面への展開

v79ではホームを「束の表紙」、その先の画面を「穴のない実用ノート紙」として統一した。ルーズリーフの穴はトップだけに残し、操作画面では横幅を優先する。

- ルーティン一覧: 一件ずつ独立した紙片。通し・パート・分析・編集を押せるタブとして明示
- ルーティン編集: 各ステップを持ち上げられる紙片にし、縦長ハンドルを青灰の綴じ代として接続
- 通し・パート練習: 現在技と正方形動画を、寸法が変わらない見開きメモとして最上部に固定
- 分析: 集計値をスタンプ、失敗率を罫線上の計測、注意点を追記メモとして整理
- 履歴: 一回の通しを青灰の見出し線で束ね、事象とメモを後書きとして表示
- 技ライブラリ: 青灰タブ付きの索引カード
- 音源ライブラリ: 錆色タブ付きの索引カード
- 設定・ヘルプ・モーダル: 同じ紙、インク、罫線、ボタンの物理反応へ統一

タイムラインだけは時間の長さと区間の判読性を優先し、紙の目盛りを重ねない単純なゲージへ戻す。編集画面の並び替えグリップも、色付きボタンではなく無地の持ち手とする。

ボタンは主要・補助とも輪郭と短い落ち影を持ち、押下時に2px沈む。チップ型は危険度やA/Bなど、状態を表す要素だけに限定する。

## 通し練習の開始導線

v81では「練習日としてのセッション開始」と「1本ごとの通し開始」を分離し、v82では確認画面から待ち時間を直接調整できるようにした。v84では本日の何本目かを開始前から実施中まで大きく表示し、累計値も開始ボタンで確認できる。

- 画面最上部に常時「通し練習モード」と表示し、パート練習や編集との取り違えを防ぐ
- 各通しは大きなオレンジのスタートボタンから、確認シート、全画面カウントダウンの順に開始する
- カウントダウンはルーティンごとに0〜15秒で設定し、既定値は5秒。通し開始の確認画面で−/＋から変更できる
- 「本日2本目」のような通し番号をスタートボタン・確認画面・全画面カウントダウン・通し実施中で一貫して表示し、開始ボタンにはこれまでの合計本数も併記する
- スタート前は失敗・クリーンを記録できず、誤タップによる通し数の混入を防ぐ
- クリーン/完走ボタンは技一覧の下へ置き、「通す → 必要なら失敗箇所 → 最後に結果」の読み順に合わせる

カウントダウンは罫紙の素材を保ちながら、離れた位置から読める大きな数字とする。0では「START」へ切り替え、楽曲がある場合は先頭からの再生を試みる。

開始ボタンは朱色の面と白文字、濃い朱の縁・落ち影で「押せる操作」として明確にする。開始後は白地へ反転し、朱色を状態表示として残す。

v86では、同じ秒数設定が複数箇所にある迷いをなくすため、右上の機能設定から開始カウントダウンを外した。秒数は実際に通しを始める確認画面でだけ調整する。

## 表示言語

v89では設定の先頭に「表示言語 / Language」を置き、日本語と英語をその場で切り替えられる。選択は端末内へ保存し、次回起動時にも維持する。

- 翻訳対象: 見出し、ボタン、補足、設定、分析ラベル、確認シート、トースト、ヘルプ
- 翻訳しないもの: 利用者が入力したルーティン名、技名、楽曲名、条件メモ、振り返り
- 英語版も日本語版と同じルーズリーフ構造・情報量を維持し、言語変更で機能配置を変えない

## サンプルのバージョン履歴

v85では、構成を編集すると分析対象が新しい版へ分かれることを初回から理解できるよう、付属サンプルに実体のある3版を用意する。

- v1「基本構成」: 基本技とフィニッシュをつないだ短い構成
- v2「移行を追加」: 持ち替えと後半の5ボールを追加
- v3「A/B分岐を追加」: 当日の調子で技を選べる分岐を追加した現行構成

40本のサンプル通し記録もv1・v2・v3へ分け、分析画面の選択欄で版ごとの結果が切り替わる。既存端末の移行では、自動生成した履歴だけを再配置し、本人が記録した通しは削除しない。

## スキン機能

v80ではスキン選択をUIと動作から外し、生成り罫紙・墨色インク・青灰タブ・錆色音源タブのルーズリーフデザインだけに固定する。

過去データに保存済みのテーマ値は破棄しないが、表示には適用しない。和・サイバー・マット白の検討内容も再検討できるよう保留し、現在の操作画面には出さない。

画面を紙で埋め尽くすのではなく、操作と意味が一致する場所だけに紙片の立体感を使う。
